# Sistema CAPEX & OPEX — TI (B&B)

Repositorio del sistema de gestión de CAPEX y OPEX para el área de Informática.

Stack: React 18 + TailwindCSS + Recharts (frontend), FastAPI + PostgreSQL 15 (backend), integración con Microsoft Dynamics NAV 2017/2018 vía OData V3, SSO con Azure AD (Office 365), despliegue on-premise con Docker Compose.

La implementación completa se incorpora vía Pull Request.
